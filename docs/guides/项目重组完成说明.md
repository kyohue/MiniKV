## 🚀 如何使用重组后的项目

### 快速开始

```bash
# Windows - 运行 CLI
scripts\build_cli.bat

# Linux/Mac - 运行 CLI
chmod +x scripts/build_cli.sh
./scripts/build_cli.sh
```

### 查找文档

所有文档都在 `docs/` 目录下：

```bash
# 使用指南
docs/guides/CLI_使用指南.md
docs/guides/中文乱码解决方案.md
docs/guides/项目结构说明.md

# 学习文档
docs/phases/Phase1_README_Guide.md

# 规划文档
docs/planning/后续阶段CLI功能规划.md
docs/planning/CLI功能演进对比.md
```

### 清理构建产物

```bash
# Windows
scripts\clean.bat

# Linux/Mac
chmod +x scripts/clean.sh
./scripts/clean.sh
```
